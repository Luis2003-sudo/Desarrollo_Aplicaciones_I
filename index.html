¿Qué es Git?

Git es una herramienta que permite controlar y guardar los cambios que se realizan en un proyecto. Es muy útil para los programadores porque ayuda a llevar un historial ordenado del código, permitiendo saber qué modificaciones se hicieron, cuándo se hicieron y quién las realizó. Además, si ocurre un error, Git permite regresar a una versión anterior del proyecto.

¿Qué es GitHub?

GitHub es una plataforma en línea que permite guardar, compartir y administrar proyectos de programación. Funciona junto con Git, ya que permite subir los archivos de un proyecto a internet y acceder a ellos desde cualquier lugar. También facilita el trabajo en equipo, porque varias personas pueden colaborar en un mismo proyecto, revisar cambios y mejorar el código.
